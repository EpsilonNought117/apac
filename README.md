# libapac  

**libapac** is a high-performance single-threaded arbitrary precision arithmetic library written in C, designed for 64-bit Windows

## Features  
- Supports both **static** and **shared (DLL)** builds  
- Uses **MSVC** as the compiler and **MASM64** as the assembler
- Optimized for 64-bit AMD/Intel CPUs 
- Provides customizable memory allocation functions  
- Detects CPU features at run-time and dispatches to best Implementation for current CPU
