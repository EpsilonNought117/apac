# libapac  

**libapac** is a high-performance arbitrary precision arithmetic library written in C, designed for Windows and built using **MSVC** and **MASM64**.  

## Features  
- Supports both **static** and **dynamic (DLL)** builds  
- Uses **MSVC** as the compiler  
- Optimized for AMD64 and x86-64 architecture  
- Provides customizable memory allocation functions  
- Automatically detects CPU features at run time
